CSS

Conceito

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo usada para definir a aparência de documentos HTML e XML. Em outras palavras, o CSS é responsável pela formatação visual das páginas web, determinando como cores, fontes, layouts e outros elementos serão exibidos.

O que é?

Apresentação visual: O CSS permite que você personalize a aparência de um site, tornando-o mais atraente e agradável aos olhos do usuário. Separação de conteúdo e estilo: Ao usar CSS, você separa o conteúdo do HTML da sua apresentação, o que torna o código mais organizado, fácil de manter e reutilizável. Design responsivo: O CSS é fundamental para criar sites responsivos, que se adaptam a diferentes tamanhos de tela (como celulares, tablets e desktops). Consistência visual: Com o CSS, você pode aplicar estilos consistentes em todo o seu site, criando uma identidade visual única. Como o CSS funciona? O CSS funciona através de regras que definem como os elementos HTML devem ser exibidos. Essas regras consistem em: Seletor: Define qual elemento HTML será afetado pelas regras. Declaração: Contém as propriedades CSS que serão aplicadas ao elemento selecionado (como cor, tamanho da fonte, etc.) e seus respectivos valores. As regras CSS podem ser aplicadas de três maneiras: Inline: Dentro das tags HTML, usando o atributo style. Embutido: Dentro da tag "style" em um documento Html. Externo: Em um arquivo CSS separado, que é vinculado ao documento HTML.
